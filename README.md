# Cucumber go to step

## Description
A Visual Studio Code extension that provides go to implementation cucumber steps on feature files.

## How to use
![](https://i.imgur.com/5crMfTj.gif)

## Installation

 - You will need to install Visual Studio Code 1.17.1 or higher.
